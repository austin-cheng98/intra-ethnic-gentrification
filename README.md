The quantitative analysis was conducted in Python 3 using the packages requests, pandas, numpy, scipy, and statsmodels. To reproduce the results, install these dependencies, obtain a free U.S. Census API key, set it as the CENSUS_API_KEY environment variable, and run the script found in this repository.

The script downloads ACS 5-year tract-level data from the U.S. Census API, constructs the analytic panel for the four study neighborhoods, applies entropy balancing to pre-treatment covariates, estimates weighted Difference-in-Differences treatment effects, and runs pre-treatment parallel-trends diagnostics. It saves the resulting tables to the output/ directory, including did_results.csv, parallel_trends.csv, matching_balance_revised.csv, and matching_weights_revised.csv.
